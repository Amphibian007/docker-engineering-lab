# Project 02 | Node.js REST API in Docker

## Overview

This project focuses on building, containerizing, and running a Node.js REST API using Docker.

The goal is to understand how a backend application moves from local development into a containerized environment while learning Docker fundamentals through practical implementation.

## Project Goals

* Build a simple Node.js REST API.
* Create a Docker image for the application.
* Run the application inside a Docker container.
* Understand container networking and port mapping.
* Manage application configuration using environment variables.
* Learn Docker debugging and troubleshooting workflows.
* Explore production-oriented container practices.

## Architecture

The application will follow this basic flow:

```text
Client
  |
  |
Node.js REST API
  |
Docker Container
  |
Docker Engine
  |
Host Machine
```
