# Docker Projects

## Overview

This section contains hands-on projects built to understand Docker through practical implementation.

Each project focuses on applying Docker concepts in real development scenarios, starting from simple containerized applications and gradually moving toward production-style architectures.

The goal is not only to build working applications, but also to understand how Docker is used in real engineering environments.

## Project Structure

Each project follows a consistent structure:

```text
project-name/

├── README.md          # Project overview and documentation
├── application/       # Application source code
├── docker/            # Docker-related files
├── docs/              # Project-specific documentation
└── diagrams/          # Architecture diagrams
```
